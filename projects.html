Fury Fly in the Withes' kitchen (2022) A super short and fun game, made at a gamejam in about 8 hours. The game was made in collaboration with Leah, Luuk and Thijs Play it here: Fury Fly.

LiFi (2022) With ever growing global internet usage, the electromagnetic spectrum is getting full. There's need for new, secure, high-speed data transfer technologies. LiFi delivers this by using light to transmit data. In this project, I made a proof-of-concept using Arduino's and a laser to transmit small amounts of data. The setup utilizes manchester encoding, frames and error detection to archive the best transfer speed, this is programmed in Arduino c++ and Python and can be viewed on Github. This project was also my PWS (NL), which is the final school research project in Holland. In the paper we researched what parameters led to the best transfer speed.

Pictionairy (2021) This site built using Express and Socket.io is a version of pictionairy but with a twist, instead of one word, three words have to be drawn: a location, an action and a person. With four players that create endless combinations, this online game is lots of fun. The source code can be found on Github.

Bayver (2019) In a collaboration with Luuk Noverraz and Midas Scheffers, Bayver was created. An adventurous route-planning app that suggests locations to visit. Just state your startpoint, endpoint and duration and Bayver generates a route passing locations that suite your preferences. Try a development version of bayver on Github.

Phillips Hue remote (2017) With this simple site, made in collaboration with Jurre and Midas, you can control your Hue lights, choose a color with the sliders, pick a preset or create your own presets. Or use disco mode if you feel like dancing!
